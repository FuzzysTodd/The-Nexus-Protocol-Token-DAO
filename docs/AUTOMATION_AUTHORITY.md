# Automation Authority

This document records the repository automation authority granted for The Nexus Protocol workspace.

## Authorized Activities

- code changes inside the repository
- documentation generation and updates
- repository scans for pre-errors, withdrawal paths, and placement opportunities
- generation of markdown and JSON reports
- web interface updates for operator visibility
- local git preparation work for code review and commit-ready changes

## Restricted Activities

- autonomous treasury movement
- withdrawal execution without human approval
- signer rotation without approval
- automatic acceptance of placement or yield recommendations as execution instructions

## Practical Meaning

Automation may continuously inspect, repair low-risk issues, document results, and prepare repository changes. Financial execution remains human-gated even when automation identifies a promising route or detects a withdrawal opportunity.
